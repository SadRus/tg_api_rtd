Типы Tg Api
=================

Библиотека типов данных, с которыми работает Tg API.

Типы Tg Api отражают структурную организацию типов официального API Telegram.
Подробнее можно ознакомиться с `API Telegram <https://core.telegram.org/bots/api#available-types>`_.
